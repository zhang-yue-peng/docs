# 证书签名文件
## 从证书颁发机构请求证书
1. 打开“钥匙串访问”应用
![er_req_keychain](./images/cer_req_keychain.png)
2. 从证书颁发机构请求证书
![er_req_req](./images/cer_req_req.png)
3. 填写证书信息
![er_req_info](./images/cer_req_info.png)
```
    - 前两项为选填项，建议填写
    - 第三项为必填项
    - 选择保存到磁盘
```
    
4. 保存
`此处会提示是否加密，选填` 

![er_req_save](./images/cer_req_save.png)
